## Overview

This project focuses on segmenting customers based on their behavior using the Mall Customer dataset. The goal is to group customers into distinct clusters to better understand purchasing patterns and support targeted marketing strategies.

The implementation is performed using Python in Google Colab.

---

## Objective

* Group customers based on behavior
* Identify distinct customer segments
* Support data-driven business decisions

---

## Dataset

**Mall Customer Dataset**

The dataset includes:

* Customer ID
* Age
* Gender
* Annual Income
* Spending Score

---

## Tasks Performed

### 1. Data Preprocessing

* Handled missing values (if any)
* Selected relevant features for clustering
* Scaled data for better model performance

---

### 2. K-Means Clustering

* Applied K-Means algorithm to group customers
* Determined optimal number of clusters using methods like the Elbow Method
* Assigned cluster labels to each customer

---

### 3. Cluster Visualization

* Visualized clusters using scatter plots
* Differentiated customer groups based on income and spending score
* Interpreted patterns within clusters
